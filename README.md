# SHADOW PROTOCOL – Top Secret Campaign

**Built entirely with Claude AI**  
A high-fidelity Three.js cyberpunk stealth fighter campaign. Pilot the classified YF-23A Black Widow II (PAV-2 "Spider") through 10 above-top-secret missions with dynamic enemies, SAM sites, aircraft unlocks, full HUD, and persistent campaign progression — all in a single browser file.

## 🎮 How to Play
1. Click **index.html** above → **Raw** → right-click → **Save As** (or just open directly)  
2. Double-click the saved file (works offline in any browser)  

**Controls**  
WASD / Arrows = pitch & roll  
Q/E = yaw  
SHIFT (hold) = accelerate  
CTRL (hold) = brake  
F = afterburner toggle  
SPACE = fire  
TAB / 1-5 = cycle weapons  
C = deploy flares  

## ✨ Features
- 10-mission escalating campaign (threat levels 8–10)  
- 5 unlockable classified aircraft (YF-23A, Bird of Prey, A-12 Oxcart, HAVE BLUE, Tacit Blue)  
- Real-time RWR, radar lock, weapon systems & scoring  
- Dynamic radio comms and detailed debriefs (S/A/B/C/D ranks)  
- Persistent save progress (localStorage)

## 🛠️ Tech Stack
- Claude AI (full game generated from single prompt)  
- Three.js r128 + Canvas HUD  
- Single self-contained HTML file (no build, no install)
